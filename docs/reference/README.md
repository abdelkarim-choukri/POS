# Reference Documents

This folder holds source material that informed the specs in `../spec/` but is
not itself authoritative.

The AiBao Cloud documentation that was used to design the extension feature
set was originally placed here. Add the markdown files here if you have them:

- `aibaocloud_user_guide.md`
- `aibaocloud_api_detailed.md`
- `aibaocloud_api_reference.md`

These files are excluded from Claude's automatic context (via `.claudeignore`)
and from graphify's knowledge graph (via `.graphifyignore`). They're kept in
the repo for traceability but should not be cited as authoritative — the
extension spec already extracted what's relevant.
